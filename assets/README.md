# Assets

This project must not redistribute proprietary MPBT retail assets.

Use local, user-provided assets from a licensed retail installation. Keep asset outputs in one of these ignored folders:

- `original/`: direct extraction from retail files.
- `scaled/`: deterministic scaled versions.
- `enhanced/`: manually cleaned or AI-upscaled variants.

Commit only tooling, manifests, import recipes, documentation, and placeholder/open assets that are safe to redistribute.
